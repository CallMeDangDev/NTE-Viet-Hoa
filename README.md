# NTE Việt Hoá — Neverness to Everness 1.3

> Bản Việt hoá không chính thức cho game **Neverness to Everness** (CN build).  

[![Discord](https://img.shields.io/badge/Discord-Tham%20gia%20server-5865F2?style=for-the-badge&logo=discord&logoColor=white)](https://discord.gg/ARfX5gf8Wn)

---

## Yêu cầu

- Game **Neverness to Everness** (CN) đã cài đặt
- Windows 10/11 64-bit

---

## Cài đặt

### Bước 1 — Tải bản phát hành

Vào tab [**Releases**](../../releases) và tải file mới nhất.

bạn sẽ có:
```
netbios.dll
game_vi.dat
viet_font.ttf
```

### Bước 2 — Đặt file vào thư mục game

Sao chép **3 file** vào thư mục:
```
C:\Program Files\Neverness To Everness\Client\WindowsNoEditor\HT\Binaries\Win64\
```

> Đây là thư mục chứa file `HTGame.exe` (hoặc `HTGame-Win64-Shipping.exe`).

### Bước 3 — Khởi động game

Mở game bình thường. Bản Việt hoá sẽ tự động kích hoạt khi game load.

---

## Gỡ cài đặt

Xoá 3 file `netbios.dll`, `game_vi.dat`, `viet_font.ttf` khỏi thư mục `Win64/` là xong.
